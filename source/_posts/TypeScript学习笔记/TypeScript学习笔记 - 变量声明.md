## `var` 声明

```typescript
var a = 10;
```

## `let` 声明
优点：块作用域、重定义及屏蔽、块级作用域变量的获取
```typescript
let a = 10;
```

# `const` 声明
引用的值是_不可变的_
```typescript
const a = 10;
```
