Docker命令大全

发表于 2022-11-04 更新于 2023-09-18 分类于 Docker笔记

1. Docker 引擎命令

1.1 Docker version 命令

docker version : 显示 Docker 版本信息。

1.2 Docker info 命令

docker info : 显示 Docker 系统信息，包括镜像和容器数。

‍

1.3 Docker help 命令

docker help : 显示指定命令的帮助

1.4 Docker events 命令

docker events : 从服务器获取实时事件

2. Docker 仓库命令

2.1 Docker search 命令

d​ocker search : 在 Docker Hub 上搜索镜像

2.2 Docker login 命令

docker login : 登录镜像服务器

2.3 Docker logout 命令

docker logout : 退出镜像服务器

2.4 Docker pull 命令

docker pull : 下载镜像

2.5 Docker push 命令

docker push : 上传镜像或者容器到远程镜像服务器

3. Docker 镜像命令

3.1 Docker images 命令

docker images : 查看所有本地主机上的镜像

3.2 Docker rmi 命令

docker rmi : 删除一个或者多个镜像

3.3 Docker tag 命令

docker tag : 标记本地镜像，将其归入某一仓库

3.4 Docker history 命令

docker history : 查看指定镜像的创建历史。

3.5 Docker build 命令

docker build : 使用 Dockerfile 创建镜像。

4. Docker 容器生命周期命令

4.1 Docker run 命令

docker run : 创建一个新的容器并运行一个命令

‍

4.2 Docker create 命令

docker create : 创建一个新的容器但不启动它，语法同 docker run

4.3 Docker start 命令

4.4 Docker stop 命令

4.5 Docker restart 命令

4.6 Docker pause 命令

4.7 Docker unpause 命令

4.8 Docker kill 命令

4.9 Docker rm 命令
